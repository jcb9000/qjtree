Detailed Description

The JsonTreeModel class.

An implementation of the QAbstractItemModel class. This class re-implements the necessary virtual functions to display and edit model elements in a QTreeView.

JsonTreeModel::updateNode.

This function is called from setData(). It is the interface between the UI and the internal TreeNode model.

Because Qt does everything by value, we have to update the TreeNode as follows:

  • Update the values in the current TreeNode
  • Get a QJsonObject or QJsonArray object from the node's parent
  • For a QJsonObject, remove the old key/value pair and insert the new key/value pair.
  • For a QJsonArray, replace the value at index i, which is given by the row in the QModelIndex.
  • Replace the parent node's data item with the new one.
  • Traverse up the tree so that the changes are propagated to the root of the document.
Parameters
nodeThe node being updated - starts at the node updated in the UI
iThe index/row of the node being updated
oldKeyOld key of the node being updated
newKeyNew key of the node being updated
valueNew value for the node
